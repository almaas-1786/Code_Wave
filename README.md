## 📝 Overview
ScamGuard helps prevent fraud by analyzing both incoming messages and transaction details. It generates a **risk score** and provides warnings **before the user makes a payment**, focusing on patterns like urgency, new recipients, high amounts, and emotional pressure.  

Unlike existing solutions, ScamGuard is **offline, lightweight, and pre-transaction focused**.

---

## ⚡ Features
- Message pattern analysis (urgency, emotional pressure, links)  
- Payment analysis (amount, recipient, notes)  
- Combined **risk scoring**  
- Instant **pre-transaction alerts**  
- Works **offline**
